The CCSK body of knowledge

The Cloud Security Alliance (CSA), a vendor neutral industry association, established the Certificate of Cloud Security Knowledge (CCSK), which demonstrates knowledge of the leading cloud security guidance documents.

While many people take theCCSKfor itscertification value, most actually report that it helps them be

more productive inadopting the right leveland model of cloud computing.

By understanding the essential cloud and cloud security concepts and applying these in conversations on IT initiatives,consensus is reached quicker. Less time is wasted on discussion and doomed cloud initiatives.Proper cloud adoptionandgovernanceis accelerated.

Most importantly, the body of knowledge facilitates a common language around cloud and cloud security. In December 2017, version 4 of the CCSK exam was released. This training is fully aligned with CCSK v4.
