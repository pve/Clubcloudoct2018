Yesterday I had the opportunity to present at the Dutch Tooling Event.
The presentation was about how service management frameworks such as ITIL could be applied to cloud computing. We have just begun to scratch the surface on that.

The presentation has some introductory material on cloud computing and then shows how cloud computing changes the conflicts that need to be resolved by IT service management.
