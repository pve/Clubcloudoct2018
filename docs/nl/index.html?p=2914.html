50 tinten cloud

  • 50 tinten cloud

Cloud computing is niet alleen maar een technologie. Dat is het belangrijkste punt dat ik vandaag zou willen maken.

Het is ook niet alleen maar een service model

Het is vooral een manier van kijken, het is een ‘state of mind’.

(Nu kunnen we natuurlijk een grap proberen te maken over toeristen die Amsterdam als één grote geestverruimende wolk zien, maar dat zou afleiden van mijn punt.)

Tot dusver denken nog veel mensen dat er IT is zoals we die altijd hadden, en dat we dan in de toekomst allemaal overgaan naar ‘de cloud’. Dan denk je vanuit de technologie, of misschien vanuit een service model.

Maar nu ik over de afgelopen jaren ruim honderd volle dagen heb besteed in cloud workshops en de definitie van cloud computing tientallen keren heb toegelicht merk ik dat er geen simpel antwoord is op de vraag of iets ‘cloud’ is. De essentiële kenmerken van cloud zijn geen stenen tafels die van de berg omlaag worden gedragen om als enige waarheid te worden verkondigd.

Het cloud computing verhaal gaat eigenlijk alleen maar over het volgende. Naarmate IT dienstverlening meer cloud kenmerken heeft zullen de afnemers meer voordelen kunnen ervaren, maar ook nieuwe risicos. Ten tweede: er zijn vele manieren om verantwoordelijkheden en controle tussen aanbieder en afnemer te verdelen, waarbij elk voordeel zijn eigen nadeel heeft.

De pure cloud spelers gaan daar vol in, en realiseren de bijbehorende voordelen voor specifieke afnemers. Maar public cloud is echt niet voor alles de enig zaligmakende oplossing. De cloud kenmerken kunnen op alle IT worden toegepast, ook op outsourcing en interne IT. Een beetje meer cloud geeft al snel een beetje meer voordeel.

De cloud is dus niet zwart of wit, maar van alles daar tussen in. Vele tinten grijs.

Welke kleur grijs heeft de cloud die bij jou past?

23 december 2014 Peter Geen categorie @nl
